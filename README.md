# 🚀 usw

A lightweight, in-browser studio for coding and running web apps instantly with zero setup.

## ✨ Features
* **In-Browser Runtimes:** Run Python and JavaScript directly in your browser tab.
* **Premium Editor:** Powered by the Monaco Editor engine.
* **Local Storage:** Saved securely on your device—no external data tracking.

## 🛠️ Getting Started
1. **Pick a Workspace:** Choose HTML/CSS, JavaScript, or Python.
2. **Write & Test:** Click `Run` to test scripts instantly.
3. **Save Work:** Files are managed locally inside your sidebar folders.
